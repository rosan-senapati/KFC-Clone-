# KFC-Clone-
It is a Clone of KFC official Website

<h1>  Get Started </h1>

Step-1:  cd KFC_Clone

Step-2: npm i json-server react-router-dom

step-3:npm run dev

(For Adding Context Please Create a Context folder inside src folder)
